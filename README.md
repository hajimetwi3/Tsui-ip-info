# Tsui IP Info
Tsui IP Info はブラウザで動く、IP アドレス・ASN・地域・接続情報の表示ツールです。Cloudflare Worker と icanhazip から、外部から見える自分の接続情報を取得して表示します。10MB のスピードテスト機能も付属しています。

## ウェブサイト版（Cloudflare Pages）

以下URLからご利用可能です。  
[https://tsuiipinfo.pages.dev/](https://tsuiipinfo.pages.dev/)
